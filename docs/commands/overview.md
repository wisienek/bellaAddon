# Commands

| Command | Permission | Usage | Notes |
| --- | --- | --- | --- |
| `/wiadomosc <zwierze> <nick>` | (none) | Prepare message | Sends templated message |
| `/test` | `beloris.test` | `/test` | Diagnostic |
| `/plecak` | (none) | `/plecak` | Open backpack UI |
| `/zaczaruj <efekt/lista> [efekt]` | `bella.enchanter` | Enchant held item | Use `lista` to show effects |
| `/przebadaj` | `bella.enchanter.check` | `/przebadaj` | Inspect held item enchant |
| `/link` | (none) | `/link` | Link MC account with Discord |
| `/jazda <on/off>` | (none) | Toggle ride | QoL toggle |
| `/otp` | `bella.otp.use` | `/otp` | Teleport to saved point |
| `/atp` | `bella.atp.admin` | `/atp` | Admin teleport |
| `/o <msg>` | (none) | OOC chat | |
| `/rnar <zasięg> [...text]` | (none) | Range narration | |
| `/date` | (none) | Show server date | |
| `/portfel [pokaż/daj <typ> <ilość>]` | `beloris.portfel.admin` (admin ops) | Wallet actions | Uses money config |
| `/bank [wpłać/wypłać/przelej <typ> <ilosć> <gracz>]` | `beloris.bank.admin` (admin ops) | Bank actions | Requires nearby bank |
| `/wymien <typ> <ilość>` | (none) | Exchange money items | Conversion rates in `money.yml` |
| `/slap <player>` | (none) | Slap player | Fun command |

Discord (via bot):
- `/who`, `/link`, `/portfel`, `/bank`, `/narracja` (see Discord bot feature page).
