# Configuration Files

All configs live in `plugins/BellaAddon/` after first run:

- `config.yml` — main options and defaults
- `tpInfo.yml` — player OTP points (`tps.<uuid>.<name>`)
- `tpLevels.yml` — teleport levels, cooldowns, radii
- `money.yml` — banks list, personal/bank balances, conversions
- `emoji.yml` — emoji mappings
- `playerConfig.yml` — player-specific settings
- `database.yml` — MySQL connection (BaseURL, Host, Port, Database, User, Password)
- `pwd.txt` — Discord bot token (single line)

Sync these files across servers if you run multiple nodes.
