# BellaAddon Docs (Docusaurus)

## Setup

```bash
cd docs
npm install
```

## Local development

```bash
npm run start
```

## Build static site

```bash
npm run build
```

## Deploy to GitHub Pages

```bash
npm run deploy
```

## Structure

- `docs/` – documentation pages
- `src/` – UI customizations (CSS, React overrides)
- `static/` – static assets (served as-is)
- `docusaurus.config.js` – site configuration
- `sidebars.js` – sidebar structure
